# wireshark-network-analysis
Wireshark-based network packet analysis and protocol inspection project for SOC and cybersecurity learning.
# Network Traffic Analysis using Wireshark

## Objective

Capture and analyze live network traffic using Wireshark and inspect common network protocols for security monitoring and packet analysis.

---

## Tools Used

* Wireshark
* Windows Command Prompt

---

## Protocols Analyzed

* DNS
* TCP
* TLS/HTTPS
* ICMP
* ARP

---

## Activities Performed

* Captured live network packets
* Generated traffic using web browsing and ping requests
* Applied protocol-based packet filters
* Inspected DNS requests and responses
* Analyzed TCP three-way handshake
* Reviewed encrypted TLS traffic
* Observed ICMP and ARP packets

---

## Key Findings

* Observed DNS domain resolution process
* Identified TCP connection establishment packets
* Analyzed encrypted HTTPS communications
* Reviewed ICMP diagnostic traffic
* Understood ARP communication and spoofing concepts

---

## Skills Gained

* Packet Analysis
* Protocol Inspection
* TCP/IP Fundamentals
* Wireshark Filtering
* Network Monitoring
* Traffic Analysis
* Basic Threat Detection
* Security Analysis

---

## Project Outcome

Successfully performed live network traffic analysis and gained practical understanding of network communication protocols and beginner-level SOC monitoring concepts using Wireshark.
